# Glossary — Ubiquitous Language

- **Product** — товар, який можна переглядати і купувати  
- **Category** — категорія товарів (наприклад, "Ноутбуки")  
- **Brand** — бренд виробника (наприклад, "Apple")  
- **User** — зареєстрований користувач системи  
- **Order** — замовлення одного або кількох товарів  
- **OrderItem** — позиція в замовленні (Product + кількість)  
- **Payment** — оплата замовлення  
- **Status** — стан об'єкта (OPEN, COMPLETED, PAID, FAILED)  
- **Discount** — знижка на товар, відображається у ціні  
- **Image URL** — посилання на локальне або зовнішнє зображення товару
